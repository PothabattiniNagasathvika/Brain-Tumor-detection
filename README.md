# 🧠 Brain Tumor Detection

## 📌 Overview
This project detects brain tumors from MRI images using deep learning techniques.
It helps in early diagnosis by classifying images into tumor and non-tumor categories.
---
## 🚀 Features

* 🖼️ MRI Image Classification
* ⚡ Fast Prediction System
* 🌐 Web-based Interface
* 📊 Deep Learning Model Integration
---
## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* NumPy, OpenCV
* Flask / Streamlit (if used)
---

## ⚙️ Installation

```bash
pip install -r requirements.txt
```
## ▶️ Run the Project
```bash
python app.py
```
